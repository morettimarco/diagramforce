About Diagramforce

Visual Salesforce architecture diagramming tool

Create architecture diagrams, data models, process flows, org charts, Gantt charts, and UML sequence diagrams for Salesforce projects. Runs entirely in your browser — no data is sent to any server.

LLMs can author diagrams via the Diagram JSON Spec.

Built with JointJS and Salesforce Lightning Design System.
